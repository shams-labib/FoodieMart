# 🍔 FoodieHub – Fast & Fresh Food Delivery App

## Live Site URL: https://assignemnt-10-8c38a.web.app

## 🚀 About This Project

FoodieHub is a modern food delivery web application where users can explore nearby restaurants, browse trending dishes, and order their favorite meals with real-time data and an engaging experience.

---

## ⭐ Key Features

- ✅ Dynamic **Hero Slider** data loaded from MongoDB
- ✅ Full CRUD operations for managing slider/restaurant items
- ✅ **Firebase Authentication** (Google/Email Sign-in)
- ✅ Stylish design using **TailwindCSS + DaisyUI**
- ✅ Smooth **Swiper.js Slider** integration
- ✅ Interactive **Modals** for user actions
- ✅ **React Icons** for visual enhancement
- ✅ Fully responsive UI for both mobile & desktop
- ✅ Dark/Light theme support

---

## 🛠️ Tech Stack

### ✅ Frontend

- React.js (Vite)
- TailwindCSS & DaisyUI
- Swiper.js
- React Icons
- Firebase Auth

### ✅ Backend

- Node.js
- Express.js
- MongoDB + Mongoose
